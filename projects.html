Our ambitious vision for 2017 - 2018

Project 1: Machine Learning

Machine learning is an inceptive topic. The subtopics include Control Systems, Neural Networks, Probability and Schotastic Processing and Machine Vision. Along the software aspect is also the hardware to be taken into perspective. The purpose of this project is to be able to read a live heartbeat and determine if it is normal. If it is not, what condition does the rhythm potentially indicate to? Based on models available online, the system will be trained on what the graphical beats for various conditions look like. In terms of Hardware, this project involves building a system to detect live heartbeat, feed it through a series of filters to clean the reading, then look at the graphical representation to be compared on the Machine Learning algorighm.
